# 🌐 Module 01 - HTML Practice Projects

Welcome to my **Module 01 HTML Practice Repository**.

This repository contains the HTML projects I built while learning the fundamentals of web development. Every project is created using **HTML5 only** without CSS or JavaScript.

## 📂 Projects

### 🛒 E-commerce Product Page
A simple online store page with product details, features, pricing, reviews, and footer.

### 🏥 Hospital Website
A semantic HTML website featuring doctors, hospital services, health tips, appointment form, and contact information.

### 📚 Online Book Store
A bookstore website with featured books, categories, customer reviews, and contact section.

### 🐶 Pet Care Center
A pet adoption website including available pets, services, pet care tips, videos, audio, and adoption form.

### ⚽ Sports Club
A sports club website with team information, match highlights, club anthem, registration form, and contact details.
